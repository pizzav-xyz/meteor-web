# Meteor Web - Web Interface

This directory contains the web-based interface for managing Meteor Client modules.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [bun](https://bun.sh/)

### Installation

1.  Navigate to this directory:
    ```bash
    cd web
    ```
2.  Install dependencies:
    ```bash
    bun install
    ```

### Running the Development Server

```bash
bun run dev
```

This will start the development server at `http://localhost:5173`.
