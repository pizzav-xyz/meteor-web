# Web Application

This directory contains the web application for Meteor Web. It's a React-based single-page application (SPA) built with [Vite](https://vitejs.dev/) and [TanStack Router](https://tanstack.com/router/).

## How It Works

The web application fetches the `modules.json` file generated by the Gradle plugin and dynamically renders a user interface to display the module information. It provides features such as searching, filtering, and favoriting modules.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [bun](https://bun.sh/)

### Installation

1.  Navigate to this directory:
    ```bash
    cd web
    ```
2.  Install dependencies:
    ```bash
    bun install
    ```

### Running the Development Server

```bash
bun run dev
```

This will start the development server at `http://localhost:5173`.
